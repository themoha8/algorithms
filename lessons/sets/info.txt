 Множество - совокупность каких-либо (вообще говоря любых) объектов — элементов этого множества. Порядок расположения элементов произвольный. Операции над множеством: добавление элемента, проверка наличия элемента, удаление элемента и другие.

Пример:

Для добавления элементов во множество будем использовать простую хеш-функию для добавления return x % 10 (индекс по последней цифре числа). Для хранения всех значений которые находятся по одному индексу в результате коллизий (например, 137, 7, 27 и т.д) будут хранится в списке.

Перебрать все элементы множества можно за O(N+k), где N - размер массива и k - кол-во элементов в массиве.

Чтобы удалить элемент нужно сначала его найти, след. поиск занимает O(k/N) (в среднем, если элементы распределены равномерно, будет тратиться k/N действий (средняя длина списка)) , а удаление за 0(1) (присвоить на место удаляемого элемента значение последнего элемента и удалить последний элемент).
